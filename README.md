# Chatbot-NLP-Concepts

## installation

```bash
install python3.8
virtualenv venv --python=python3.8
pip install -r .\requirements.txt
pip freeze > requirements.txt
```

## create the training model
(venv)

```
python training.py
```

the model will be save in data.pth

## run the chatbot

(venv)

```
python chatbot.py
```

Now chat with your new friend !
